**InjectIt** is a compact and effective DLL injector that lets you easily insert your dynamic libraries into any target application — including games, utilities, and custom tools.

---

## 🧠 What It Offers

- Manual map and LoadLibrary injection  
- Supports 32-bit and 64-bit targets  
- Auto-detects running processes  
- Built-in error log viewer  
- Minimal UI, instant execution

---

## 🚀 How to Launch

1. Start the target application  
2. Run `Loader.exe`  
3. Select your `.dll` file  
4. Choose method and click “Inject”
